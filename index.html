Here are a few of my projects that are easy to explore, there are many more on github and linkedin:

Utility Forecasting at Duke

A dashboard built using jupyter which allows Duke University's facilities management department to forecast campus utility usage. Provides implementations of several anomaly detection and time series forecasting techniques.

Github Repo

Climate Analysis

An intuitive, visualization-based analysis of historical temperature data, exploring the efficacy of making climate predictions via statistical modeling. This is an attempt to guide the layman through a basic prediction about climatology, to try to make the whole idea of climate science more accessible.

Jupyter Notebook Github Repo

Graph Paper

A simple, open source graph paper tool that runs in the browser. Too many simple web apps have account systems and other bloated features when they don't really need them, so I thought it was cool that this project didn't have any of that. I built the upload/download functionality, and the fill tool.

Demo Github Repo

Discord Event Bot

A chatbot for scheduling and managing evens on a Discord server. Schedule events, get RSVPs from server memebers, and get reminders in your discord DMs.
